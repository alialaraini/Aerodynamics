\begin{itemize}
\tightlist
\item
  Source Distribution on an Airfoil -
\end{itemize}

    In Lesson 3 of \emph{AeroPython}, you learned that it is possible to
represent potential flow around a circular cylinder using the
superposition of a doublet singularity and a free stream. But potential
flow is even more powerful: you can represent the flow around \emph{any}
shape. How is it possible, you might ask?

For non-lifting bodies, you can use a source distribution on the body
surface, superposed with a free stream. In this assignment, you will
build the flow around a NACA0012 airfoil, using a set of sources.

Before you start, take a moment to think: in flow around a symmetric
airfoil at \(0^{\circ}\) angle of attack,

\begin{itemize}
\tightlist
\item
  Where is the point of maximum pressure?
\item
  What do we call that point?
\item
  Will the airfoil generate any lift?
\end{itemize}

At the end of this assignment, come back to these questions, and see if
it all makes sense.

    \subsection{Problem Setup}\label{problem-setup}

    You will read data files containing information about the location and
the strength of a set of sources located on the surface of a NACA0012
airfoil.

There are three data files: NACA0012\_x.txt, NACA0012\_y.txt, and
NACA0012\_sigma.txt. To load each file into a NumPy array, you need the
function
\href{http://docs.scipy.org/doc/numpy/reference/generated/numpy.loadtxt.html}{\texttt{numpy.loadtxt}}.
The files should be found in the \texttt{resources} folder of the
\texttt{lessons}.

    Using 51 mesh points in each direction, and a domain
\([-1, 2]\times[-0.5, 0.5]\), compute the velocity due to the set of
sources plus a free stream in the \(x\)-direction with \(U_{\infty}=1\).
Also compute the coefficient of pressure on your grid points.

    \subsection{Questions:}\label{questions}

    \begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  What is the value of maximum pressure coefficient, \(C_p\)?
\item
  What are the array indices for the maximum value of \(C_p\)?
\end{enumerate}

Make the following plots to visualize and inspect the resulting flow
pattern:

\begin{itemize}
\tightlist
\item
  Stream lines in the domain and the profile of our NACA0012 airfoil, in
  one plot
\item
  Velocity potential in the domain and the profile of our NACA0012
  airfoil, in one plot
\item
  Distribution of the pressure coefficient and a single marker on the
  location of the maximum pressure
\end{itemize}

    \textbf{Hint}: You might use the following NumPy functions:
\href{http://docs.scipy.org/doc/numpy/reference/generated/numpy.unravel_index.html}{\texttt{numpy.unravel\_index}}
and
\href{http://docs.scipy.org/doc/numpy/reference/generated/numpy.argmax.html}{\texttt{numpy.argmax}}
